# Фильмопоиск

- **SPA**: ветка "main"
- **Реализация на Next**: пулл-реквест из ветки "next"

### Установка и запуск проекта:

1. Клонировать репозиторий:
   `git clone`

2. Установить зависимости:
   `pnpm install`

3. Запустить сервер actors и сервер main (бэкенд)

4. Запустить проект в dev-режиме:
   `pnpm run dev`
